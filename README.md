# Freelance Hourly Rate Calculator

A simple, privacy-friendly tool to calculate your ideal freelance hourly rate in seconds.

[Live calculator 2](https://evanschaslin0-lgtm.github.io/reimagined-giggle/)  
*(Enable GitHub Pages in repo settings if this URL is not yet live.)*

## Features
- Calculates recommended hourly rate given your income goal, annual expenses, and expected billable hours.
- Responsive, minimal designworks great on desktop and mobile.
- No data saved or tracked. 100% client-side.  
- MIT Licensed and open source.

## Usage
1. Enter your annual income target, expected annual business expenses, and how many billable hours you plan to work per year.
2. Click Calculate. Your recommended hourly rate is instantly displayed!
3. Adjust numbers to explore different scenarios.
